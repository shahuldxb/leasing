# VodaLease Enterprise — Video Production Plan

## Phase 2: Global Definitions

### Visual Style Specification
- **Sub-genre**: Corporate motion graphics / tech product demo
- **Rendering + Line**: 3D glass-morphism UI panels, clean vector elements, particle effects
- **Color + Lighting**: Dark navy/charcoal (#0f172a) backgrounds, glowing teal (#06b6d4) and gold (#f59e0b) accents, soft ambient glow, rim lighting on UI elements
- **Detail density**: Moderate — clean compositions, not cluttered, premium SaaS aesthetic

### Recurring Elements
- **VodaLease UI Panels**: Floating glass-morphism cards with dark backgrounds, teal borders, white text, showing dashboard KPIs
- **Data Streams**: Flowing lines of light (teal/gold) representing data movement
- **Office Environment**: Modern glass towers, abstract cityscape silhouettes

### Voice Profiles
- **Off-screen narrator**: Male, professional/confident tone, moderate pace, English

### BGM Source Decision: Separate
- **Genre & Style**: Modern corporate ambient, uplifting tech
- **Tempo & Rhythm**: 100 BPM, steady subtle pulse
- **Key & Scale**: C major
- **Core Instrumentation**: Piano, soft synth pads, subtle electronic percussion, light strings
- **Soundscape/Ambiance**: Clean, spacious, professional
- **Production Quality**: High-quality, polished studio production
